# Jafora — אפליקציית דוח יומי

אפליקציית Android לצילום אקסל, חילוץ נתונים עם Gemini, ושליחת דוח בוואטסאפ.

## מה האפליקציה עושה

1. צילום / בחירת תמונה של האקסל
2. Gemini קורא את **התאריך האחרון עם נתונים** (באר שבע + צרעה)
3. מילוי העברות משאיות
4. יצירת דוח בנוסח קבוע
5. שליחה לוואטסאפ

## התקנה על Samsung (APK)

### שלב 1 — קבל מפתח Gemini (חינם)

1. היכנס ל-[Google AI Studio](https://aistudio.google.com/apikey)
2. צור API Key
3. שמור אותו — תזין אותו בהגדרות האפליקציה

### שלב 2 — בנה APK

```bash
cd mobile
npm install
npx eas login
npx eas build --platform android --profile preview
```

> אם אין לך חשבון Expo — צור בחינם ב-[expo.dev](https://expo.dev)

הבנייה רצה בענן (~10-15 דקות). בסוף תקבל קישור להורדת APK.

### שלב 3 — התקן על הטלפון

1. הורד את קובץ ה-APK
2. Samsung → **הגדרות → אבטחה → התקנה ממקורות לא ידועים** — אפשר
3. פתח את הקובץ והתקן

## פיתוח מקומי

```bash
cd mobile
npm install
npx expo start
```

סרוק QR עם **Expo Go** על הטלפון לבדיקות.

## לוגיקת תאריך

האפליקציה לוקחת את **התאריך האחרון** בטבלה שיש בו נתונים מלאים לבאר שבע ולצרעה (לא בהכרח היום).

## עלות

~20 תמונות/חודש = **חינם** (מכסת Gemini החינמית).
