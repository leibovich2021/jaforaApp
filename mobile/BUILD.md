# Jafora — מדריך התקנה אישית (Samsung)

אפליקציה פרטית לשימוש שלך בלבד. **בלי חנות.**

---

## חלק א' — מפתח Gemini (5 דקות)

1. פתח: **https://aistudio.google.com/apikey**
2. התחבר עם חשבון Google
3. לחץ **Create API Key**
4. העתק את המפתח (מתחיל ב-`AIza...`)
5. שמור — תזין אותו באפליקציה

> מנוי Gemini בתשלום ≠ מפתח API. צריך API Key מ-AI Studio.

---

## חלק ב' — בניית APK (פעם אחת, ~15 דק')

פתח **PowerShell** בתיקייה `mobile` והרץ:

```powershell
.\build-apk.ps1
```

הסקריפט יבקש:
1. **התחברות ל-Expo** (חינם) — פותח דפדפן, פעם אחת
2. **הגדרת פרויקט** — אוטומטי
3. **בניית APK** בענן — ~10-15 דק'

בסוף תקבל **קישור להורדת APK**.

### אם הסקריפט לא רץ:
```powershell
cd mobile
npm install
npx eas login
npx eas build:configure
npx eas build --platform android --profile preview
```

---

## חלק ג' — התקנה על Samsung

1. הורד את קובץ `.apk` מהקישור
2. שלח לטלפון (WhatsApp / email / USB)
3. **הגדרות → אבטחה → התקנה ממקורות לא ידועים** — אפשר
4. פתח את הקובץ → **התקן**
5. אייקון **Jafora** יופיע על המסך
